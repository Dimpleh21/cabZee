# CabZee - Student Carpooling Website

CabZee is a carpooling platform designed specifically for college students to coordinate and share rides seamlessly. Inspired by BlaBlaCar, it enables verified peers to find convenient ride-sharing options within their campus community.

## Features
- User-friendly frontend built with Next.js, Bootstrap, Framer Motion, AOS, and Lottie for engaging visuals and smooth animations
- Backend powered by Node.js, Express.js, and MongoDB Atlas for reliable data management and real-time updates
- Secure authentication and session management via NextAuth
- MVP functionality to create and join carpools effortlessly, helping reduce commuting costs and carbon footprint

## Tech Stack
- Frontend: Next.js, Bootstrap, Framer Motion, AOS, Lottie
- Backend: Node.js, Express.js, MongoDB Atlas
- Authentication: NextAuth

## Getting Started
1. Clone the repository
2. Install dependencies with \`npm install\` or \`yarn\`
3. Configure environment variables for MongoDB Atlas and NextAuth
4. Start the development server using \`npm run dev\` or \`yarn dev\`
5. Access the app in your browser and start sharing rides
